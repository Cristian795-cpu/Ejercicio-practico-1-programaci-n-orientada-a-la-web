## 📝 Descripción
La startup **TechGadget Store**, especializada en dispositivos electrónicos innovadores, te ha contratado para desarrollar la página de productos destacados de su tienda en línea. Este proyecto implementa un sistema de **lista de deseos (wishlist)** usando `localStorage` que permite a los usuarios guardar productos de interés antes de comprar.

## ✨ Funcionalidades Implementadas
- ✅ **Diseño Responsive**: Grid adaptable (3-2-1 columnas)
- ✅ **Wishlist con localStorage**: Persistencia de datos
- ✅ **Contador dinámico**: Actualización en tiempo real
- ✅ **Botones inteligentes**: Cambian de estado al agregar
- ✅ **Limpiar wishlist**: Botón para resetear todo
- ✅ **Sin duplicados**: Control de versiones con ramas (develop, feature, main)

## 🚀 Tecnologías Utilizadas
- **HTML5** - Estructura semántica
- **CSS3** - Grid, Flexbox, Media Queries
- **JavaScript** - Manipulación del DOM, localStorage, Event Listeners
- **Git & Git Flow** - Control de versiones
